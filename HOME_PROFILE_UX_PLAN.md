# 🏠 홈화면 & 프로필 UX/UI 개선 기획

## 📌 개요

홈화면과 프로필 페이지를 사용자 친화적으로 개선하여 **첫인상**과 **지속적 사용**을 유도합니다.

---

## 🎯 핵심 목표

1. **홈화면**: 사용자가 매일 방문하고 싶게 만드는 **감성적 터치** + **실용적 정보**
2. **프로필**: 개인화된 인사이트와 **성취감** 제공

---

## 🏠 홈화면 개선 계획

### 1. 인사말 섹션 (Header)
**현재**: 단순한 "안녕하세요, {이름}님!"
**개선**:
- 시간대별 인사 (아침/점심/저녁/밤)
- 날씨 기반 감성 문구 (선택적)
- 오늘 날짜 표시

**예시**:
- "좋은 아침이에요, {이름}님! ☀️" (06:00-11:59)
- "안녕하세요, {이름}님! 🌤️" (12:00-17:59)
- "좋은 저녁이에요, {이름}님! 🌙" (18:00-22:59)
- "안녕하세요, {이름}님! 🌃" (23:00-05:59)

### 2. 프로필 완성도 배너
**현재**: ProfileCompletionBanner 컴포넌트 사용
**개선**: 유지 (이미 구현됨)

### 3. 오늘의 분석 CTA (Main CTA)
**현재**: 기본적인 그라데이션 버튼
**개선**:
- 더 큰 시각적 임팩트
- 애니메이션 효과 (hover, active)
- 분석 횟수 표시 (예: "오늘 1회 분석")
- 마지막 분석 후 경과 시간 표시 (예: "3일 전 분석")

**디자인**:
- 그라데이션 배경 (pink-500 → purple-500)
- 아이콘 애니메이션 (펄스 효과)
- 그림자 강화

### 4. 최근 분석 결과 카드
**현재**: 기본적인 카드
**개선**:
- 더 큰 썸네일
- 주요 지표 요약 (색소, 모공, 주름 등)
- 개선 추이 표시 (이전 분석과 비교)
- "다음 분석 추천일" 표시

**디자인**:
- 이미지 오버레이 (분석 점수 배지)
- 호버 효과 강화
- 더 나은 타이포그래피

### 5. 피부 상태 요약 (새로 추가)
**목적**: 한눈에 보는 피부 상태 개요
**내용**:
- 전체 피부 점수 (0-100)
- 주요 문제 영역 (Top 3)
- 개선 추이 (최근 7일/30일)

**디자인**:
- 미니 차트 (라인 또는 바)
- 색상 코딩 (좋음/보통/주의)

### 6. 추천 시술 섹션
**현재**: RecommendedTreatments 컴포넌트
**개선**:
- 더 나은 카드 디자인
- "왜 추천되는지" 간단한 이유 표시
- 관심 등록 버튼 (선택적)

---

## 👤 프로필 페이지 개선 계획

### 1. 프로필 헤더
**현재**: 기본적인 아바타 + 이름
**개선**:
- 더 큰 아바타 (프로필 사진 업로드 기능 추가 가능)
- 사용자 등급/뱃지 (예: "분석 마스터", "꾸준한 사용자")
- 가입일 표시

### 2. 통계 카드 (ProfileStats)
**현재**: 분석 횟수, 개선 추이 등
**개선**:
- 더 나은 시각화 (차트, 그래프)
- 비교 통계 (평균 대비)
- 성취 배지

### 3. 빠른 액션 (QuickActions)
**현재**: 기본적인 버튼들
**개선**:
- 더 나은 아이콘
- 액션별 설명 추가
- 진행 상태 표시

### 4. 프로필 정보 상세
**현재**: 기본적인 정보 표시
**개선**:
- 섹션별 구분 강화
- 편집 가능한 필드 표시
- 개인정보 보호 안내 강화

### 5. 분석 기록
**현재**: 최근 분석 기록 리스트
**개선**:
- 더 나은 카드 디자인
- 필터링 옵션 (날짜, 점수)
- 정렬 옵션

---

## 🎨 디자인 시스템 적용

### 색상
- Primary: Pink-500 → Purple-500 (그라데이션)
- Success: Green-500
- Warning: Amber-500
- Error: Red-500

### 타이포그래피
- 헤더: text-2xl, font-bold
- 본문: text-base, font-normal
- 캡션: text-sm, text-gray-600

### 간격
- 섹션 간격: space-y-6
- 카드 내부: p-6
- 요소 간격: gap-4

### 그림자
- 카드: shadow-lg
- 버튼: shadow-md (hover: shadow-lg)

---

## 📱 반응형 디자인

- 모바일 우선 (max-w-md)
- 태블릿: max-w-2xl
- 데스크톱: max-w-4xl
